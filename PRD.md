# LetsZoom - Product Requirements Document

## 1. Executive Summary

**LetsZoom**는 프레젠테이션, 교육, 기술 데모를 위한 화면 확대 및 주석 도구입니다. Microsoft의 ZoomIt과 유사한 기능을 제공하면서, 현대적인 UI/UX와 추가적인 기능을 통해 사용자 경험을 향상시킵니다.

## 2. Product Vision

"모든 발표자와 교육자가 청중의 이해를 돕기 위해 쉽게 화면을 확대하고 주석을 추가할 수 있는 직관적인 도구"

## 3. Target Users

- **Primary**: 프레젠터, 교육자, 기술 강사
- **Secondary**: IT 전문가, 소프트웨어 데모 진행자, 콘텐츠 크리에이터

## 4. Core Features (MVP)

### 4.1 화면 확대 (Screen Zoom)
- **단축키 기반 확대**: 사용자 정의 가능한 단축키로 화면 확대/축소
- **부드러운 확대**: 애니메이션 효과를 통한 자연스러운 확대
- **확대 배율 조절**: 마우스 휠 또는 키보드로 1.2x ~ 20x 배율 조절
- **확대 중 이동**: 마우스로 확대된 화면 내 이동
- **초점 영역 표시**: 확대할 영역을 시각적으로 표시

### 4.2 화면 그리기 (Drawing & Annotation)
- **기본 도구**:
  - 펜/브러시: 자유 곡선 그리기
  - 직선: Shift 키로 직선 그리기
  - 화살표: 방향 표시
  - 사각형: 영역 강조
  - 원/타원: 원형 영역 표시
  - 텍스트: 화면에 텍스트 추가
  - 하이라이터: 반투명 강조 표시

- **그리기 옵션**:
  - 색상 선택: 8가지 기본 색상 + 커스텀 색상
  - 선 두께: 얇음(2px), 보통(4px), 굵음(8px), 매우 굵음(16px)
  - 투명도 조절: 불투명 ~ 반투명
  - 실행 취소/다시 실행 (Ctrl+Z / Ctrl+Y)
  - 모두 지우기 (ESC 또는 'E' 키)

### 4.3 스크린샷 캡처
- **캡처 모드**:
  - 전체 화면 캡처
  - 영역 선택 캡처
  - 주석이 포함된 캡처

- **저장 옵션**:
  - 클립보드로 복사
  - 파일로 저장 (PNG, JPG)
  - 자동 파일명 생성 (날짜/시간 기반)
  - 저장 경로 설정

### 4.4 전역 단축키
- **기본 단축키**:
  - `Ctrl + 1`: 화면 확대 모드
  - `Ctrl + 2`: 그리기 모드 (확대 없음)
  - `Ctrl + 3`: 확대 + 그리기 모드
  - `Ctrl + 4`: 스크린샷 캡처
  - `ESC`: 모든 모드 종료

- **단축키 커스터마이징**: 설정에서 모든 단축키 변경 가능

### 4.5 설정 UI
- **일반 설정**:
  - Windows 시작 시 자동 실행
  - 시스템 트레이에 상주
  - 알림 설정

- **확대 설정**:
  - 기본 확대 배율
  - 확대 애니메이션 속도
  - 확대 시 화면 경계 처리

- **그리기 설정**:
  - 기본 색상 및 두께
  - 펜 스타일
  - 터치/펜 입력 지원 활성화

- **단축키 설정**:
  - 모든 단축키 커스터마이징
  - 단축키 충돌 감지

## 5. Advanced Features (Post-MVP)

### 5.1 라이브 드로잉 (Live Draw)
- 확대 없이 전체 화면에 반투명 오버레이로 그리기
- 응용 프로그램 위에 그리기 가능

### 5.2 화면 녹화
- 확대/주석 과정을 비디오로 녹화
- MP4, AVI 형식 지원

### 5.3 프리셋 저장
- 자주 사용하는 설정을 프리셋으로 저장
- 프리셋 간 빠른 전환

### 5.4 다중 모니터 지원
- 모니터별 독립적인 확대/주석
- 모니터 선택 UI

### 5.5 클라우드 동기화
- 설정 및 프리셋 클라우드 동기화
- 여러 PC에서 동일한 설정 사용

## 6. Technical Requirements

### 6.1 성능
- 확대 응답 속도: < 100ms
- 그리기 입력 지연: < 16ms (60fps)
- 메모리 사용량: < 100MB (대기 중)
- CPU 사용률: < 5% (대기 중), < 20% (확대/그리기 중)

### 6.2 호환성
- **OS**: Windows 10 이상
- **화면 해상도**: 1920x1080 ~ 3840x2160 (4K)
- **입력 장치**: 마우스, 터치스크린, 디지털 펜

### 6.3 보안 및 개인정보
- 로컬 실행, 네트워크 연결 불필요 (클라우드 기능 제외)
- 캡처된 이미지는 사용자 지정 폴더에만 저장
- 사용 통계 수집 옵트인

## 7. User Experience

### 7.1 첫 실행 경험
1. 시스템 트레이 아이콘 표시
2. 간단한 튜토리얼 표시 (스킵 가능)
3. 기본 단축키 안내
4. 첫 번째 확대/그리기 시 화면 가이드 표시

### 7.2 사용 흐름
1. 백그라운드에서 대기
2. 단축키 입력
3. 확대/그리기 모드 활성화
4. 작업 수행
5. ESC로 종료 또는 자동 종료

### 7.3 시각적 피드백
- 모드 전환 시 화면 모서리에 작은 인디케이터
- 현재 도구 및 색상을 보여주는 미니 툴바
- 단축키 힌트 (옵션)

## 8. Success Metrics

### 8.1 정량적 지표
- 다운로드 수
- 일일 활성 사용자 (DAU)
- 평균 사용 시간
- 기능별 사용 빈도

### 8.2 정성적 지표
- 사용자 만족도 (설문조사)
- 사용 편의성 점수
- 버그 리포트 수
- 기능 요청 수

## 9. Release Plan

### Phase 1: MVP (v1.0) - 4-6주
- 화면 확대
- 기본 그리기 도구
- 스크린샷 캡처
- 단축키 설정
- 기본 설정 UI

### Phase 2: Enhancement (v1.5) - 2-3주
- 고급 그리기 도구
- 라이브 드로잉
- 프리셋 기능
- UI/UX 개선

### Phase 3: Advanced (v2.0) - 3-4주
- 화면 녹화
- 다중 모니터 지원
- 클라우드 동기화
- 플러그인 시스템

## 10. Risks & Mitigation

### 10.1 기술적 리스크
- **화면 캡처 성능 이슈**: GPU 가속 사용, 최적화된 렌더링
- **단축키 충돌**: 충돌 감지 및 경고, 대체 단축키 제안
- **고해상도 화면 지원**: DPI 인식, 적응형 렌더링

### 10.2 경쟁 리스크
- **ZoomIt의 무료 제공**: 추가 기능 및 더 나은 UX로 차별화
- **유사 제품 존재**: 현대적 디자인 및 사용자 친화적 경험
